# atmospheric-parameters-web-app
🌍 AtmoSphere
Real-Time Atmospheric Parameters Monitoring System

“Visualizing the Air Around You — Instantly, Intelligently.”

📌 Overview

AtmoSphere is a modern, software-based web application that provides real-time atmospheric data for any city worldwide. It offers a clean, intuitive interface with interactive visualizations, helping users easily understand weather conditions.

🚀 Features
🌡 Real-Time Temperature Monitoring
💧 Humidity & Wind Speed Tracking
🌅 Sunrise & Sunset Information
📊 Interactive Temperature Graph
📅 5-Day Weather Forecast
⭐ Add to Favorites System
🔍 Smart City Search
🌙 Light/Dark Mode Toggle
📍 Location-based Weather Detection
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
API: OpenWeatherMap API
Deployment: GitHub & Vercel
⚙️ How It Works
User searches for a city or enables location
Application sends request to weather API
API returns real-time atmospheric data
Data is processed and displayed on UI
Charts and forecasts are generated dynamically
